Política de Privacidade

Última atualização: Janeiro de 2025 · Versão 1.0

🔒 Resumo: O CryptoEdge Pro é uma plataforma auto-hospedada. Seus dados ficam no seu próprio servidor VPS. Não coletamos, não vendemos e não compartilhamos seus dados com terceiros.

1. Informações Gerais

O CryptoEdge Pro ("Plataforma", "Software") é um sistema de software para day trade de criptomoedas fornecido como plataforma auto-hospedada (self-hosted). Esta Política de Privacidade descreve como os dados são tratados no contexto de uso do software.

Ao utilizar a Plataforma, você confirma que leu e concorda com esta política.

2. Dados Coletados

2.1 Dados de Autenticação

  • Nome de usuário e senha (armazenados com hash SHA-256 + salt no seu servidor)
  • Tokens de sessão (expiram em 30 dias, armazenados localmente)

2.2 Dados Operacionais

  • Histórico de trades registrados pelo usuário
  • Configurações da plataforma e preferências
  • Histórico de análises do Analysis AI
  • Alertas de preço configurados
  • Chaves de API (Binance, Telegram) — armazenadas apenas no seu servidor

2.3 Dados de Terceiros (Opcional)

  • Binance API: Preços em tempo real (público), dados de conta (opcional, requer suas chaves)
  • TradingView: Widget de gráfico (pode coletar dados conforme política deles)
  • API de IA (laozhang.ai): Mensagens enviadas ao chat de IA Expert (quando configurado)
  • Telegram: Notificações (somente se configurado pelo usuário)

3. Armazenamento e Segurança

⚡ Importante: Todos os dados são armazenados exclusivamente no servidor VPS do operador. O desenvolvedor do software não tem acesso a esses dados.

  • Banco de dados: SQLite local em /data/cryptoedge.db
  • Senhas: hash SHA-256 com salt único
  • Tokens: gerados com 256 bits de entropia
  • Transmissão: HTTPS/TLS quando configurado com Nginx + Let's Encrypt
  • Chaves de API: armazenadas em variáveis de ambiente, nunca expostas ao frontend

4. Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Autenticação e controle de acesso à plataforma
  • Exibição do histórico e estatísticas do usuário
  • Funcionamento das funcionalidades (alertas, bot, análises)
  • Personalização da experiência (tema, configurações)

5. Compartilhamento de Dados

Não compartilhamos seus dados com terceiros, exceto nas seguintes situações de serviços que você escolhe configurar:

  • Binance: para executar operações (se você configurar suas chaves de API)
  • Telegram: para enviar notificações (se você configurar o bot)
  • API de IA: para responder mensagens do chat IA Expert (se você configurar)

6. Direitos do Usuário

Como os dados ficam no seu próprio servidor, você tem controle total:

  • Acesso: Todos os dados estão no arquivo cryptoedge.db
  • Exclusão: Delete o arquivo de banco de dados para remover todos os dados
  • Portabilidade: Use a função de exportação de relatório (JSON/CSV)
  • Correção: Edite diretamente via interface ou banco de dados

7. Cookies e Armazenamento Local

A plataforma utiliza localStorage do navegador para:

  • Token de autenticação (sessão do usuário)
  • Preferências de tema (claro/escuro)
  • Credenciais de login lembradas (somente se o usuário marcar "Lembrar acesso")

Não utilizamos cookies de rastreamento, pixels de marketing ou tecnologias de rastreamento de terceiros.

8. Menores de Idade

A plataforma é destinada exclusivamente a pessoas maiores de 18 anos. Não coletamos intencionalmente dados de menores.

9. Aviso de Risco Financeiro

⚠️ Aviso importante: Day trade e operações alavancadas em criptomoedas envolvem alto risco de perda. O CryptoEdge Pro é uma ferramenta de apoio e não constitui recomendação de investimento. O operador e desenvolvedor não se responsabilizam por perdas financeiras resultantes do uso da plataforma.

10. Alterações desta Política

Esta política pode ser atualizada. A versão mais recente estará sempre disponível em /privacy na sua instância da plataforma.

11. Contato

Para questões relacionadas à privacidade ou uso do software, entre em contato com o operador da sua instância da plataforma.
